# AI Learning Assistant – Product Roadmap

This document describes the long-term vision of the app.
Only Version 1 is implemented in code. Other versions represent future growth.

---

## v1 – Core Learning Engine (Portfolio MVP)
- Paste text content
- Generate:
  - Simplified explanation
  - Explain like I’m 5
  - Key points
- Ask follow-up questions
- Clean UI
- Domain-driven architecture
- Server Actions + AI integration

---

## v2 – Knowledge History
- Save KnowledgeUnits
- View past lessons
- Delete / edit units
- Basic search

---

## v3 – Quiz System
- Generate quizzes from KnowledgeUnits
- Score answers
- Track correctness

---

## v4 – Learning Progress
- Learning streaks
- Topic completion
- Weak areas detection

---

## v5 – Smart Search
- Semantic search using embeddings
- Ask:
  “What did I learn about React state?”

---

## v6 – File Upload
- Upload PDF / DOC
- Extract content
- Generate KnowledgeUnits from files

---

## v7 – Audio Learning
- Convert explanations to audio
- Listen while walking or driving

---

## v8 – Visual Learning
- Generate diagrams
- Simple concept illustrations

---

## v9 – Sharing & Collaboration
- Share KnowledgeUnits
- Public learning pages
- Community learning

---

## v10 – Personal AI Teacher
- Full personalized learning plans
- Adaptive difficulty
- Topic recommendations
- Memory of user weaknesses
