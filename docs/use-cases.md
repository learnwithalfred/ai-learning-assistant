# AI Learning Assistant – Use Cases

## 1. Create KnowledgeUnit
**Actor:** User  
**Goal:** Learn a piece of content  

**Flow:**
1. User pastes content
2. Clicks **Teach me**
3. AI returns:
   - Simplified explanation
   - Child-friendly explanation
   - Key points
4. App stores the KnowledgeUnit

---

## 2. View KnowledgeUnit
**Actor:** User  

**Flow:**
1. User sees a list of KnowledgeUnits
2. Clicks one
3. App shows explanations and key points

---

## 3. Ask Follow-Up Questions
**Actor:** User  

**Flow:**
1. User types a question
2. AI answers using the KnowledgeUnit content

---

## 4. Generate Quiz
**Actor:** User  

**Flow:**
1. Clicks **Quiz me**
2. AI generates questions
3. User answers
4. App evaluates responses

---

# Examples

## Example 1: Benefits of Brushing Teeth

**Flow:**
1. User asks: *Why do I need to brush my teeth?*
2. Clicks **Teach me**
3. AI returns:
   - **Simplified explanation:**  
     Brushing keeps your teeth clean, stops cavities, and keeps your gums healthy.
   - **Child-friendly explanation:**  
     Brushing your teeth is like cleaning your toys. If you don’t, germs make your teeth sad and sick!
   - **Key points:**
     - Keeps teeth clean
     - Prevents cavities
     - Keeps gums healthy
4. Stored in history as: *Benefits of Brushing Teeth*

---

## Example 2: Multiplication

**Flow:**
1. User asks: *What is 7 × 8?*
2. Clicks **Teach me**
3. AI returns:
   - **Simplified explanation:**  
     If you have 7 groups of 8 apples, you have 56 apples in total.
   - **Child-friendly explanation:**

   - **Key points:**
